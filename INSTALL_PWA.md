# 📱 Как установить LS STUDIO как приложение

## iOS (iPhone/iPad)

### Safari
1. Откройте `mobile.html` в Safari
2. Нажмите кнопку **"Поделиться"** (квадрат со стрелкой вверх)
3. Прокрутите вниз и нажмите **"На экран «Домой»"**
4. Введите название (например, "LS STUDIO")
5. Нажмите **"Добавить"**

Приложение появится на главном экране и будет открываться **без адресной строки Safari** в полноэкранном режиме!

---

## Android (Chrome)

### Chrome
1. Откройте `mobile.html` в Chrome
2. Нажмите **три точки** (меню) в правом верхнем углу
3. Выберите **"Добавить на главный экран"**
4. Подтвердите название
5. Нажмите **"Добавить"**

Или внизу может появиться плашка **"Установить приложение"** — нажмите на неё.

Приложение появится в меню приложений и на главном экране!

---

## Что даёт установка

✅ **Полноэкранный режим** — без адресной строки браузера  
✅ **Иконка на главном экране** — быстрый доступ  
✅ **Работа офлайн** — Service Worker кэширует файлы  
✅ **Быстрый запуск** — приложение загружается мгновенно  
✅ **Как нативное приложение** — выглядит и ощущается как родное приложение

## Технические детали

### Файлы PWA
- `manifest.json` — конфигурация приложения (название, иконки, цвета)
- `sw.js` — Service Worker для кэширования и офлайн-работы
- `mobile.html` — обновлённая версия с meta-тегами для PWA

### Meta-теги
```html
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
<link rel="manifest" href="manifest.json">
```

### Display mode: standalone
В `manifest.json` указано `"display": "standalone"` — это открывает приложение без интерфейса браузера (адресной строки, кнопок навигации).

---

## Требования

- **HTTPS** или **localhost** (для тестирования)
- Современный браузер (Safari 11.3+, Chrome 67+)
- iOS 11.3+ или Android 5+

> ⚠️ **Важно:** Service Worker работает только через HTTPS (или localhost для разработки).

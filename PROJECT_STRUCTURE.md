# LS STUDIO EKB - Структура проекта

## 📁 Структура файлов

```
LightStream/
├── index.html              # Десктопная версия
├── mobile.html             # Мобильная версия (app-подобный дизайн)
│
├── css/
│   ├── styles.css          # Стили десктопной версии
│   └── mobile/
│       └── mobile.css      # Стили мобильной версии
│
├── js/
│   ├── app.js              # Логика десктопной версии (Vue 3)
│   └── mobile/
│       └── mobile-app.js   # Логика мобильной версии (Vue 3)
│
├── api/
│   └── api.js              # API mock для интеграции
│
├── scrn/                   # Скриншоты интерфейса
│   ├── ref/                # Референсы (Сбербанк)
│   │   ├── lght/           # Светлая тема
│   │   └── drk/            # Тёмная тема
│   └── ...                 # Другие скриншоты
│
└── html/                   # Оригинальные файлы для референса
```

## 📱 Версии сайта

### Десктопная (`index.html`)
- Боковое навигационное меню
- Табличное представление данных
- Полноэкранные таблицы
- Развёрнутые карточки статуса

### Мобильная (`mobile.html`)
- App-подобный дизайн (референс: Сбербанк Онлайн)
- Нижняя навигационная панель (tab bar)
- Плитки/карточки в 2 колонки
- Крупные tap-зоны (44x44px минимум)
- Закруглённые углы (16-20px)
- Поиск в шапке
- Переключатель языка

## 🎨 Дизайн-система (Mobile)

### Цвета
- Фон: `#f5f5f7`
- Карточки: `#ffffff`
- Текст основной: `#1c1c1e`
- Текст вторичный: `#8e8e93`
- Акцент: `#dc5f20` (оранжевый)
- Успех: `#34c759` (зелёный)

### Типографика
- Заголовки секций: 22px, 700
- Заголовки карточек: 17px, 600
- Значения: 24px, 700
- Подписи: 13px, 400
- Навигация: 11px, 500

### Отступы
- Секции: 20px 16px
- Карточки: 16px
- Сетка: gap 12px
- Закругления: 16-20px

## 🚀 Запуск

### Десктоп
Откройте `index.html` в браузере

### Мобильная версия
Откройте `mobile.html` в браузере (или в режиме эмуляции мобильного)
